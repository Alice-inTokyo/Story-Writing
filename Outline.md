# 1. 基础信息 (Basic Info)
**书名备选：** 《网恋对象是不可名状邪神》、《男友失踪后，深渊祂降临了》、《怪物他自卑了》

**预估字数：** 45万 - 50万字（约 150 章）

**核心标签：** 赛博朋克 / 微克苏鲁 / 人外 / 救赎 / 甜虐交织 / 破镜重圆

**一句话简介：** 那个在网上自卑又黏人的完美男友，真身竟然是不可名状的高维神明？

# 2. 核心设定与背景 (World & Background)
## A. 关键概念
高维生物 (Z：Zero)：诞生于宇宙缝隙的古老生命，形态不可名状（触手/星云/眼球），自带精神污染。

本源能量注入（核心补丁）：7岁那场事故中，男主的一丝本源能量意外注入女主身体。这导致了女主成为唯一的“锚点”，也是两人能精神链接、甚至共享生命的生物学基础。

认知滤网：互联网是两者交流的唯一缓冲区，过滤掉了致命的 San 值污染。

## B. 往事真相（虐点根源）
起因：反派实验室贪婪地试图汲取高维能量，导致实验室崩塌。

事故：7岁的女主误入缝隙，见到了狂暴边缘的男主。男主并非有意伤害，但能量失控。

牺牲：女主父母为保护女儿，强行切断连接并因此牺牲。

误会：男主认为是自己的存在害死了女主父母，因此极度自卑、自厌，时刻准备自我毁灭以赎罪。

结果：女主因能量冲击导致解离性障碍（情感阻断/失忆），男主则躲在深渊中默默守护。

# 3. 人物小传 (Characters)
### 男主： 祁源 (Zero)
**身份：** 亿万年的孤独神明 / 卑微的恋爱初学者。

外貌：

人类伪装：苍白俊美，带有病态易碎感，总是把自己包裹得很严实。

真实形态：流淌的深蓝星云，无数幽光触手，核心是璀璨星辰。

性格：美强惨、社恐、极致自卑、自我攻略。

核心矛盾：深爱女主，却觉得自己是杀害她父母的凶手；渴望触碰，又怕污染她。

### 女主：纪零 (Ji Ling)
身份：天才数据修复师 / 情感缺失症患者 / 神明的唯一锚点。

外貌：清冷挂，眼神永远平静无波，像精致的人偶。

性格：绝对理性、钝感力满级、清醒的疯批（后期）、护短。

特殊能力：

情感阻断：无恐惧，无痛觉（直到被救赎）。

万物互联：因体内有神明能量，能操控大部分电子设备和低级异常生物。

恋爱观：“他是怪物？不，根据数据分析，他是世界上最完美的伴侣。”
